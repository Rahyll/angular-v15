üìñ NTX Select Component Guide

The NTX Select component is a powerful, customizable dropdown/select component built for Angular applications. It provides advanced features like multiple selection, confirmation mode, search functionality, and excellent performance with large datasets.

üöÄ Installation & Setup

1. Import the Module

import {{ '{' }} NtxSelectModule {{ '}' }} from './ntx-select/ntx-select.module';

@NgModule({{ '{' }}
  imports: [
    NtxSelectModule,
    // ... other imports
  ]
{{ '}' }})
export class AppModule {{ '{' }} {{ '}' }}

2. Basic Usage

<ntx-select
  [(ngModel)]="selectedValue"
  [options]="options"
  placeholder="Select an option">
</ntx-select>

‚öôÔ∏è Input Properties

Property Type Default Description
options NtxSelectOption[] [] Array of options to display in the dropdown
placeholder string '' Placeholder text when no option is selected
multiple boolean false Enable multiple selection mode
confirmation boolean false Show Apply/Cancel buttons for multiple selection
required boolean false Mark the field as required for validation
disabled boolean false Disable the select component
searchable boolean true Enable search functionality within options
clearable boolean true Show clear button to reset selection

üì° Output Events

Event Type Description
selectionChange EventEmitter<any> Emitted when selection changes (single value or array for multiple)
optionSelect EventEmitter<NtxSelectOption> Emitted when an individual option is selected
optionDeselect EventEmitter<NtxSelectOption> Emitted when an individual option is deselected
dropdownOpen EventEmitter<void> Emitted when dropdown opens
dropdownClose EventEmitter<void> Emitted when dropdown closes

üí° Usage Examples

Single Selection

<ntx-select
  [(ngModel)]="selectedCountry"
  [options]="countryOptions"
  placeholder="Select your country"
  (selectionChange)="onCountryChange($event)">
</ntx-select>

Multiple Selection

<ntx-select
  [(ngModel)]="selectedCategories"
  [options]="categoryOptions"
  [multiple]="true"
  placeholder="Select categories"
  (selectionChange)="onCategoriesChange($event)">
</ntx-select>

With Confirmation

<ntx-select
  [(ngModel)]="selectedItems"
  [options]="itemOptions"
  [multiple]="true"
  [confirmation]="true"
  placeholder="Select items (Apply to confirm)">
</ntx-select>

Reactive Form Integration

<form [formGroup]="myForm">
  <ntx-select
    formControlName="mySelect"
    [options]="options"
    [required]="true"
    placeholder="Required field">
  </ntx-select>
</form>

‚úÖ Best Practices

üéØ Performance

  • Use virtual scrolling for datasets with 100+ items
  • Implement proper change detection strategies
  • Consider lazy loading for very large datasets
  • Use OnPush change detection when possible

‚ôø Accessibility

  • Always provide meaningful placeholder text
  • Use proper ARIA labels and roles
  • Ensure keyboard navigation works correctly
  • Provide clear error messages for required fields

üì± Responsive Design

  • Test on various screen sizes
  • Use appropriate container widths
  • Consider mobile touch interactions
  • Implement proper overflow handling

üîß Form Integration

  • Use reactive forms for complex validation
  • Implement proper error handling
  • Provide clear user feedback
  • Handle form state changes appropriately
